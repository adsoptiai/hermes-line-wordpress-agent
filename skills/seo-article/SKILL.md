# SEO Article Skill

## Output Requirements

- Create drafts by default.
- Include a clear H1/title, meta title, meta description, slug, excerpt, categories, tags, and image brief.
- Respect site profile limits for title and description length.
- Include internal link suggestions when the site profile provides target pages.
- Do not keyword-stuff.

## Workflow

1. Confirm topic, audience, search intent, and language.
2. Draft outline.
3. Generate article body.
4. Generate SEO fields.
5. Optionally generate/upload a featured image.
6. Create WordPress draft.
7. Send preview URL to LINE.

## Publish Rule

Publishing or scheduling must be a separate approved action after the user reviews the preview.
